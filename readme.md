# Students of Goju Ryu Website

A responsive website for a traditional Goju Ryu Karate dojo in Sweden, built with vanilla HTML, CSS, and JavaScript.

## Purpose

The site provides information about training programs (Karate, Arnis, Yoga), schedules, and locations for prospective and current members across three locations: Trosa, Mariefred, and Lund.

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Inter, Raleway)
